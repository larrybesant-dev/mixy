# Soft-Launch Live Checklist

## Success Rule
- no user asks what is going on

## Preflight
- room opens
- host controls visible
- moderation controls ready
- observer ready to take notes

## Join Check
- all users appear correctly
- chat is visible
- mic status is understandable
- nobody is duplicated or missing

## Chaos Check
- refresh works
- reconnect works
- rejoin works
- host transfer works
- kick and rejoin recovery works

## Trust Check
- no one seems confused about mic ownership
- chat never feels broken
- audio behavior feels natural
- moderator actions feel fast and reliable

## Stop Immediately If
- users ask what is happening
- ownership feels wrong
- people vanish or duplicate visibly
- chat or audio becomes untrustworthy

## Go Decision
Proceed to the next phase only if the room feels clear, stable, and trustworthy to real users.
