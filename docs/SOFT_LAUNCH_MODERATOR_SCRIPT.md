# Soft-Launch Moderator Script

## Before Opening
- confirm the room can be ended quickly
- confirm you can remove a disruptive user
- confirm chat and host controls are visible

## Opening Script
Say this at the start:

Welcome. Please use the room normally.
If anything feels confusing, delayed, or strange, say it immediately.
Do not wait for a bug to become big.

## During Session
### Minute 0 to 5
- let everyone join
- confirm roster, chat, and mic state look correct

### Minute 5 to 15
- normal conversation
- pass the mic between users

### Minute 15 to 25
- ask one tester to refresh
- ask one tester to drop and rejoin
- observe without over-explaining

### Minute 25 to 40
- test host transfer
- test kick and rejoin recovery

### Minute 40 to 60
- let the room run naturally
- step in only if something breaks

## If Something Breaks
- end the session immediately if trust or usability drops sharply
- do not try to push through a broken experience
- log the time, device, and exact user quote

## Debrief Questions
Ask every tester:
1. Did anything feel confusing?
2. Did you ever not know what was happening?
3. If this was a real app, would you trust it?
